# AI Model Evaluation Results

This repository contains evaluation results from multiple AI models performing image analysis tasks.

## Repository Structure

### Data
- **`data/`** - Contains the images used for evaluation

### Evaluation Results

Additional reviews have been performed and stored in the following folders:

#### Claude 4.5 Results
- **`revision_11_25/`** - Contains Claude 4.5 evaluation results
  - `results/` subfolder includes reasoning and aggregated results for 10 repetitions

#### OpenAI Results
- **`revision_11_25_OpenAI/`** - Contains OpenAI model evaluation results (GPT-4o and GPT-5-mini)
  - `results/` subfolder includes reasoning and aggregated results for 10 repetitions

#### Gemini Results
- **`revision_11_25_Gemini/`** - Contains Gemini model evaluation results
  - `results/` subfolder includes reasoning and aggregated results for 10 repetitions

### Original Submission
- **`original_submission/`** - Contains results from the original submission

### Overall Results
- **`overall_results/`** - Contains aggregated results across all models

## Methodology

Each model was evaluated with 10 repetitions to ensure consistency and reliability of results. The results folders contain:
- Individual reasoning traces for each repetition
- Aggregated statistical results across all repetitions
